# 交通費支給単価シミュレーター 仕様・計算根拠説明書

本シミュレーターは、最新のエネルギー価格に基づき、各車種別の1kmあたりの適切な交通費支給単価を算出するためのツールです。

---

## 1. 計算の基本方針
本ツールでは、経済産業省が発表するガソリン価格および、電気料金目安単価を基準とし、以下のポリシーで算出を行っています。

- **小数点処理**: すべての最終算出単価は、小数点第1位を**四捨五入**し、1円単位の整数とします。
- **現行比較**: 2023年改訂版の社内規定（または一般的な基準値）との差額をリアルタイムに算出します。

---

## 2. 車種別 計算根拠

### 2.1 自動車（ガソリン車）
- **計算式**: `レギュラーガソリン価格(円/L) ÷ 基準燃費(km/L)`
- **デフォルト基準燃費**: **15.4 km/L**
  - ※国土交通省の平均燃費データおよび実燃費の乖離を考慮した係数。

### 2.2 バイク（125cc超）
- **計算式**: `レギュラーガソリン価格(円/L) ÷ 基準燃費(km/L)`
- **デフォルト基準燃費**: **22.8 km/L**

### 2.3 EV（電気自動車）
- **計算式**: `新電力料金目安単価(円/kWh) ÷ 基準電費(km/kWh)`
- **デフォルト基準電費**: **7.4 km/kWh**
  - ※国内主要EVの平均的な交流電力量消費率に基づく。

### 2.4 PHEV（プラグインハイブリッド）
- **計算式**: `(自動車の算出単価 + EVの算出単価) ÷ 2`
- **考え方**: ガソリン走行と電気走行が混在することを想定し、両ブランチの単純平均を採用しています。

---

## 3. データソース（基準値）
シミュレーターのデフォルト値は、以下の公的または信頼性の高いデータを参照しています。

- **レギュラーガソリン価格**: 経済産業省 資源エネルギー庁「石油製品価格調査」
- **電気料金目安単価**: 全国家庭電気製品公正取引協議会「新電力料金目安単価」
  - 現在の基準：**31円/kWh** (税込)

---

## 4. プログラム構成・技術スタック
本ツールは、可搬性とパフォーマンスを両立するため、単一の静的Web技術で構成されています。

- **Frontend**: HTML5 / JavaScript (Vanilla JS)
- **Styling**: Tailwind CSS (UIフレームワーク)
- **Animation**: GSAP (数値カウントアップ・フェードアウト等)
- **Chart**: Chart.js (過去12ヶ月の推移グラフ)
- **Deployment**: GitHub Pages

---

## 5. 運用上の注意点
- 基準燃費および電費は、設定パネルより任意に変更が可能です。自社の車両構成に合わせて調整を行ってください。
- 本計算結果はあくまでシミュレーションであり、最終的な支給額の決定は各社の社内規定に準じます。
