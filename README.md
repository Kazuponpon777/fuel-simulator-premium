# 交通費支給単価シミュレーター Premium

[![GitHub Pages](https://img.shields.io/badge/deploy-GitHub%20Pages-blue?style=flat-square&logo=github)](https://Kazuponpon777.github.io/fuel-simulator-premium/)

エネルギー価格（ガソリン・電気）の変動に基づき、適切な交通費支給単価をリアルタイムでシミュレーションするダッシュボードツールです。

---

## ✨ 主な機能
- **リアルタイム計算**: ガソリン価格や電気料金を入力すると、即座に全車種の単価を算出。
- **現行比較**: 既存の社内規定との差額を自動計算し、増減をバッジで表示。
- **基準値のカスタマイズ**: 車種ごとの基準燃費・電費をユーザーが任意に調整可能。
- **価格推移グラフ**: 過去12ヶ月のガソリン価格推移をビジュアル化。
- **プレミアム・デザインシステム**:
    - **グラスモフィズム**: 透過とブラーを活かした先進的なUI。
    - **ダークモード採用**: システム設定に連動し、目に優しいダークテーマを提供。
    - **滑らかなアニメーション**: 数値カウンタやUI遷移による心地よい操作体験。
